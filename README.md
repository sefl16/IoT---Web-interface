# Startup

## Setup NPM dependencies:
* Stand in Frontend map "/Frontend" 
* NPM install

## Setup python script:
* Pip3 install Flask
* Pip3 install Flask_restful
* Pip3 install Flask_cors


## Start python script:
* Python3 main.py

## Start database from PHP
* Stand in project directory
* php -S 127.0.0.1:8080 -t .\backend
